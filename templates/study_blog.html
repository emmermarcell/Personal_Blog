Physics Information and References

A collection of great books concerning my favorite fields in physics:

Special and General Relativity:
  • The Geometry of Minkowski Space Time by Gregory L. Naber
  • General Relativity by Robert M. Wald
Particle Physics:
  • Introduction to Elementary Particles by David J. Griffiths
Quantum Field Theory:
  • An Introduction to Quantum Field Theory by Michael E. Peskin and Daniel V. Schroeder
  • Quantum Field Theory by Mark Srednicki
Group Theory:
  • Group Theory in Physics by Wu-Ki Tung
  • Group Theory and its Application to the Quantum Mechanics of Atomic Spectra by Eugene Wigner
  • Introduction to Group Theory with Applications by Gerald Burns

Machine Learning References

Some references I found useful for studying machine learning. I recommend to go through the books in the order they are listed.

Books:

  • "The Hundred-Page Machine Learning Book" by Andriy Burkov
  • "Hands-On Machine Learning with Scikit-Learn, Keras, and TensorFlow" by Aurélien Géron
  • "Deep Learning" by Ian Goodfellow, Yoshua Bengio, and Aaron Courville
  • "Natural Language Processing with Transformers" by Lewis Tunstall, Leandro von Werra, and Thomas Wolf

Courses:

  • Andrew Ng's Machine Learning Course on Coursera
  • Practical Deep Learning for Coders on fast.ai
  • The NLP, Deep RL and Audio Courses on Hugging Face

Other programming references

Here are some books that I found useful for studying programming, mostly Python:

  • Effective Computation in Physics by Anthony Scopatz and Kathryn D. Huff
    A practical guide to using computational tools in physics research.
  • Python Crash Course, 3rd Edition by Eric Matthes
    A hands-on introduction to programming with Python.
  • Serious Python by Julien Danjou
    A guide to writing high-performance, maintainable Python code.
